🧠 Cuckoo Hashing
Two hash functions, two tables. Each key has exactly 2 possible slots.
h1(key) = key % TABLE_SIZE    h2(key) = (key // TABLE_SIZE) % TABLE_SIZE
If a slot is taken, the old bird gets evicted (cuckoo style!) and must find its alternate slot.
How to Play: 1️⃣ Click a bird card to select it.   2️⃣ Click the correct nest (shown in green) to place it.   Wrong nest = penalty!   If a nest is occupied, the current bird evicts the old one — just like real Cuckoo Hashing!   Complete all birds to advance levels. 🏆
